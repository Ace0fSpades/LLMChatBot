# LLM ChatBot Frontend

Frontend приложение для LLM ChatBot на React + TypeScript.

## Технологии

- **React 18** - UI библиотека
- **TypeScript** - типизация
- **Redux Toolkit** - управление состоянием
- **React Router** - маршрутизация
- **Vite** - сборщик
- **Axios** - HTTP клиент

## Структура проекта

```
frontend/
├── src/
│   ├── components/      # React компоненты
│   │   ├── auth/        # Компоненты аутентификации
│   │   ├── chat/        # Компоненты чата
│   │   ├── sidebar/     # Компоненты боковой панели
│   │   └── common/       # Общие компоненты
│   ├── hooks/           # React хуки
│   ├── services/       # API сервисы
│   ├── stores/         # Redux store
│   ├── types/          # TypeScript типы
│   ├── utils/          # Утилиты
│   └── styles/         # Стили
├── public/             # Статические файлы
└── dist/              # Собранные файлы (генерируется)
```

## Установка и запуск

### Разработка

```bash
# Установить зависимости
npm install

# Запустить dev сервер
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

### Сборка для production

```bash
# Собрать проект
npm run build
```

Собранные файлы будут в папке `dist/`. Backend автоматически подхватит их и будет раздавать статические файлы.

## Конфигурация

Создайте файл `.env` на основе `.env.example`:

```env
VITE_API_URL=http://localhost:8080
```

## Функциональность

- ✅ Регистрация и авторизация пользователей
- ✅ Управление чатами (создание, переключение, архивирование)
- ✅ Потоковая передача ответов через SSE
- ✅ Сохранение истории сообщений
- ✅ Адаптивный UI

## Интеграция с Backend

Frontend собирается в `dist/` и подхватывается backend при запуске в production режиме. Backend раздает статические файлы и обрабатывает SPA роутинг.
